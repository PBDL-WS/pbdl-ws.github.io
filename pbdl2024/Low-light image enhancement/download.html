Content of The Downloaded Package

In the dataset, you may find a folder for training data and a folder for testing data.

Train
In the training dataset folder, participants will find two subfolders named "normal" and "low." The "normal" folder contains images taken under normal-light conditions, while the "low" folder contains images taken under low-light conditions. The images in both folders are named with pure numerical values, and corresponding images with the same name establish a relationship. Each image is an sRGB image with a resolution of 6720x4480, and there are a total of 1700 pairs of images.

Validation
The validation dataset consists of 70 pairs of images, each cropped to 1120x1120 size . The format of the validation images is consistent with the training set, stored in '.png' files. These files include input low-light sRGB images, output normal-light sRGB images. However, the ground truth normal-light images are not available to participants.

Test
For the test dataset, there are 100 sRGB images provided as input, each with a spatial resolution of 1120x1120. Similar to the other datasets, these images are available in '.png' format. However, the test dataset does not include the ground truth normal-light images.
