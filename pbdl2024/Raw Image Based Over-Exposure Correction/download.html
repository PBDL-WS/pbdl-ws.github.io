Content of The Downloaded Package

RPO dataset should follow folder structure bellow:

Train
The training data set consists of 300 gt and corresponding overexposed images (ratio=3,5,8,10). The resolutions of RAW images and corresponding sRGB images are both 6744 x 4502.

Validation
The validation set processes RAW images into four-channel (RGGB) images, crops them and saves them as .mat files. Unlike the training set, the validation set only includes input files and does not have ground truth

Test
Like the validation set, the test set also crops RAW images and saves them as .mat files without ground truth.

If you use this dataset for any purpose, please cite our previous work
